# Минималистичный блог на PHP

Проект представляет собой простой блог с возможностью регистрации, авторизации, создания постов, загрузки изображений и комментированиев.

## Содержимое проекта

- **index.php** – главная страница со списком постов
- **login.php** – страница входа
- **register.php** – страница регистрации
- **create.php** – создание нового поста
- **post.php** – просмотр поста и комментариев
- **logout.php** – выход из аккаунта
- **css/style.css** – стили сайта
- **includes/** – вспомогательные файлы
- **data/** – JSON-файлы с данными
- **uploads/** – загруженные изображения

## Функционал

- Регистрация
- Авторизация
- Создание постов с текстом и изображениями
- Комментирование постов

## Языки

- PHP
- HTML
- CSS
- JSON
